Chart.js with React | react-chartjs-2

Chart.js is a JavaScript library that can be used to create charts and graphs with ease. ReactJS is JavaScript library used for building reusable UI components.

Using Chart.js along with ReactJS allows us to create multiple charts using components and dynamic data.


Chart.js can be used to create 8 types of charts:
Line, Bar, Radar, Doughnut & Pie, Polar Area, Bubble, Scatter, Area, and Mixed.

Pre-requisites:

Installation of ReactJS& NodeJS.


Start a project using

npx create-react-app graphs

Change directory to the project graphs

cd graphs

Start the server using the command

npm start

INSTALLATION:

We will install react-chartjs-2 which is a React wrapper for Chart.js

npm install --save react-chartjs-2 chart.js

Alternatively, download the latest version of Chart.js from Github or use CDN links in script tags.
